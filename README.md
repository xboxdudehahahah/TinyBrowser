# TinyBrowser
The lightest browser on Android (lighter than naked browser)

Information:
This browser is pretty barebones but dosen't have useless features,
it dosen't have history so it is a good privacy feature,you can search any website! I might do the lightest android game sometimes in the future (possibly snake)
The app normally is 3kb but i'm forced to sign the app so it's 12Kb
Virustotal link:https://www.virustotal.com/gui/file/2bda77b97a17938660928943f904ad82cdcb0c51cac7ecb71a5ecb049ad16fef/summary
This project is fully open source (FOSS) !
The app support every languages that is on your phone, and to go back to the homepage turn your device, i know it's pretty weird but it's better
